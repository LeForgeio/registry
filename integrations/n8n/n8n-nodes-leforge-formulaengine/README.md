# n8n-nodes-LeForge-formulaengine

n8n community node for LeForge formula-engine.



## Installation

### Community Nodes (Recommended)

1. Go to **Settings** > **Community Nodes**
2. Click **Install**
3. Enter `n8n-nodes-LeForge-formulaengine`
4. Agree to the risks
5. Click **Install**

### Manual Installation

```bash
npm install n8n-nodes-LeForge-formulaengine
```

Then restart n8n.

## Configuration

1. Add LeForge credentials:
   - Go to **Credentials** > **Add Credential**
   - Search for "LeForge"
   - Enter your API Key and Base URL

2. Use the node in your workflows
